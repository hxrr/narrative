# @narrative/babel-plugin-compiler

Performance enhancement compiler for `Narrative`.

## Usage

```bash
npm install @narrative/babel-plugin-compiler
```

.babelrc:

```json
{
  "plugins": ["@narrative/compiler"]
}
```
